## Overview

* Mini Text Toolkit (MTT) is an open source offline text conversion tool. This tool was designed to provide an aid in solving crypto CTF challenges. But it also can be used for various other purpose üòÉ.

## Pre-requisites

* Python 3
* A module named pysimplegui. You can easily install it with `pip install pysimplegui`.

## Installation

* Curl:
`curl https://github.com/shafdo/Mini-Text-Toolkit -O mtt.py`

* Clone the repo
`git clone https://github.com/shafdo/Mini-Text-Toolkit`

___

* Linux
`python3 mtt.py`

* Windows
`python mtt.py`
